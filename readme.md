

## Social Listening & Intent Analysis

*Tools that monitor the web (Reddit, Twitter/X, LinkedIn) to find people actively looking for your solution.*

  - [Agent-Reach](https://github.com/Panniantong/Agent-Reach) - MCP-compatible CLI that lets AI agents read and search Twitter, Reddit, YouTube, and GitHub without paid API keys.
  - [Awario](https://awario.com) - Brand monitoring and social listening tool that tracks mentions across the web and social networks.
  - [Buska](https://buska.io) - Mentions monitoring with sentiment analysis across social media and community platforms.
  - [F5Bot](https://f5bot.com) - Free keyword alert service for Reddit, Hacker News, and Lemmy.
  - [GummySearch](https://gummysearch.com) - Audience research tool for Reddit that surfaces communities and high-intent conversations around a topic.
  - [Leado](https://leado.co) - AI-powered Reddit lead generation tool that monitors subreddits for high-intent leads and writes personalized replies automatically.
  - [Scweet](https://github.com/Altimis/Scweet) - Python library for scraping tweets, profiles, followers, and following lists from Twitter/X without an API key.
  - [Syften](https://syften.com) - Keyword monitoring across Slack communities, Discourse forums, Reddit, and other platforms in real time.
  - [twikit](https://github.com/d60/twikit) - Async Python client for Twitter's internal API enabling tweet search, profile scraping, and timeline access without paid credentials.
  - [twscrape](https://github.com/vladkens/twscrape) - X/Twitter scraper with multi-account pooling that covers search results, user profiles, followers, and tweet metadata.
  - [URS](https://github.com/JosephLai241/URS) - Universal Reddit Scraper for archiving subreddits, comment threads, and redditor data via the PRAW API.
